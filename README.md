# Human Following Robot

## Description
Human Following Robot is an intelligent system that automatically tracks and follows a person using sensors and control algorithms. It maintains a safe distance and avoids obstacles while moving.

## Features
- Automatically follows a human
- Obstacle detection and avoidance
- Maintains safe distance
- Real-time movement tracking

## Components Used
- Arduino / Microcontroller
- Ultrasonic Sensor
- IR Sensor / Camera
- Motor Driver (L298N)
- DC Motors
- Power Supply

## Working Principle
The robot detects a human using sensors or camera input. Based on the detected position, it moves forward, backward, or turns to follow the person while avoiding obstacles.

## Applications
- Personal assistance
- Shopping carts
- Surveillance systems
- Smart robotics

## Future Improvements
- Add AI-based face recognition
- Voice control
- Mobile app integration
